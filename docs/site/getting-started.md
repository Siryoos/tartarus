# Getting Started with Tartarus

This guide will help you set up Tartarus and deploy your first sandbox.

## Prerequisites

- Linux or macOS
- Docker (for local development)
- Go 1.21+

## Installation

```bash
# Install Tartarus CLI
curl -sL https://get.tartarus.io/install.sh | bash
```

## Running a Sandbox

```bash
tartarus run --image ubuntu:latest --name my-first-sandbox
```
